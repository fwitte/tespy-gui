# Drag and Drop Graphical User Interface for TESPy

This is a proof of concept GUI for TESPy, still very experimental and only
providing the topology of the system. THe goal is to be able to create,
parametrize and to execute TESPy models from the GUI in the future.

Clone the repository and install it with uv:

    cd ./path/to/clone
    uv sync

Then run from terminal

    tespy-gui
